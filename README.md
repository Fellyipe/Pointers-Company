## 📌 Sobre o projeto

Este repositório apresenta um protótipo de serious game 2D desenvolvido como trabalho de conclusão de curso, com o objetivo de apoiar o ensino de Estruturas de Dados.

O jogo foi implementado no GameMaker Studio 2 e propõe a representação de conceitos abstratos — como ponteiros, nós, listas encadeadas e vetores — por meio de puzzles interativos e elementos visuais.

O projeto não se limita à implementação: envolve levantamento de requisitos com docentes da área de Computação, definição de mecânicas alinhadas a objetivos pedagógicos e avaliação com usuários (professores e estudantes).
